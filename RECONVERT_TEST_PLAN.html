1. Introduction

This document defines the master test plan and detailed test cases for the ReConvert (Upsell.com) Shopify embedded application. ReConvert enables merchants to customize post-purchase experiences including Thank You pages, Order Status pages, and Product page upsell slots through a visual Flow Builder with pre-made widgets (Banner, Product recommendations, etc.).

Testing focuses on app authentication, dashboard navigation, Flow Builder editor behavior, widget configuration, save/persistence, preview rendering, and storefront-facing upsell display on the Shopify test store.

2. Test Objectives

  • Verify merchants can access the app via signed Shopify URL and open Flow Builder without errors.
  • Validate all Banner pre-made widget controls (status, content, toggles, button, save).
  • Validate Product recommendations widget — Product details block and Success/Error banner configuration.
  • Confirm configuration persists after Save and page reload.
  • Ensure embedded iframe UI is stable at desktop viewport (1600×1000).
  • Detect regressions in collection picker, slot management, and widget deletion flows.

5.1 Preconditions

  • Valid signed app URL with non-expired JWT token for the test shop.
  • Test shop has sample products (e.g., "The Compare at Price Snowboard") and "Automated Collection".
  • Chrome/Chromium installed for automation (playwright install chromium).
  • Network access to app.reconvert.com and Shopify CDN.

6. Entry & Exit Criteria

Entry Criteria

  • Test build deployed or staging URL available.
  • Valid test shop credentials / signed URL provided.
  • Automation suite dependencies installed.

Exit Criteria

  • 100% of P0 test cases pass.
  • ≥ 95% of P1 test cases pass; remaining documented with workaround.
  • No open Critical/Blocker defects.

13. Smoke Test Checklist (5 minutes)

  • Signed app URL loads — Edit flows visible
  • Flow Builder opens — welcome message in iframe
  • Thank you page → Banner widget settings accessible
  • Banner status dropdown works
  • Product page → Product recommendations editor opens
  • Save button enabled and clickable
  • No uncaught JavaScript errors in console

  • TC-A01, TC-A02 (App access)
